# eduWeb
xx教育网站系统

### 项目说明

该系统可用于中小型教育网站建设  
前端HTML + CSS + jQuery  
后端thinkPHP + MySQL

#### 功能说明

- 支持用户登录、注册账号。
- 用户可以学习教师发布的课程，或者通过BBS与他人进行讨论。
- 教师可以通过markdown语法在线发布课程。
- 用户可以在个人空间发布心得体会，与世界分享自己的想法。
- 支持头像上传，个人信息自定义，让你变得与众不同。
- ...更多功能等待你的发现！
